### Added

* Assert: Return error value from [`assert.rejects()`]({% link api/assert/rejects.md %}) and [`assert.throws()`]({% link api/assert/throws.md %}), for example to run other assertions against your rejection value. (Bartosz Dziewoński) [#1824](https://github.com/qunitjs/qunit/issues/1824)
* CLI: Add support for [`--reporter`]({% link cli.md %}#--reporter) with local script. (Jordan Kasper, Timo Tijhof) [#1823](https://github.com/qunitjs/qunit/issues/1823)
* Core: Introduce [`QUnit.config.ignoreUnhandledRejections`]({% link api/config/ignoreUnhandledRejections.md %}). (Timo Tijhof) [#1829](https://github.com/qunitjs/qunit/issues/1829)
